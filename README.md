# Doky

Doky é um projeto pessoal feito com o objetivo de facilitar a administração, padronização e curva de aprendizado da seção administrativa a qual trabalho.

## Observação

Não há **NENHUMA** informação sensível neste projeto.

## Dificuldade observada

Existem centenas de processos e cada um deles tem uma ordem `específica` de tarefas a serem executadas. Isto faz com que um novo usuário tenha uma curva de aprendizado muito alta e tenha muita insegurança nas suas tomadas de decisões.

O diferencial é a possibilidade de, ao cadastrar um processo, fazer a vinculação dele com um _Tracker_. Isso significa que quando o usuário clicar em _Edit_ haverá uma seção _Tracker_ onde será mostrado o que ele deve fazer para concluir aquele processo e caso ele não saiba como escrever aquele documento ele poderá se basear em documentos cadastrados na seção _Modelos_.

Portanto, com o **Tracker** e **Modelos** um novo usuário tende a conseguir finalizar os processos com uma maior facilidade e confiança.
